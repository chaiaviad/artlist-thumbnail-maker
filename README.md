# Artlist Tools

דף נחיתה פנימי שמפצל לכלים של Artlist. הכל רץ בדפדפן, ללא שרת.

- **index.html** — דף הנחיתה (hub).
- **thumbnail-banner-maker.html** — Thumbnail & Banner Maker.
- **collab-maker.html** — Collab Maker (לוגו Artlist + שותף, מרווחי גייד מדויקים, scrim מרוכך, snap + Ctrl+Z, ייצוא PNG שקוף).
- **Guide/** — נכסי הגייד הרשמי (לוגו + דוגמאות מיקום).
